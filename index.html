ABOUT DIREWOLF

DireWolf Kernel is a new kernel and it works on both Redmi 4A and 5A. It is developed to give the maximum in every aspect of the device. We added governors, i/o schedulers, various other things to get the maximum out of the device. It supports spectrum profiles and also has a lot of other stuffs that tweakers like.

How To Flash :

  • Download the kernel zip from the link below
  • Boot into recovery mode
  • Select DireWolf-vxx.zip file
  • Flash the zip Reboot
